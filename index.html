A Public Programming Performance

About

Public Programming Performance is an in-class, live collaborative programming experiment where a few volunteer students will code at the front of the class. Using VSCode Live Edit and a shared GitHub repo, these volunteers will write code live while collaborating with classmates. Volunteers can seek assistance from their peers during the session and have the option to tag out, allowing another student to take over their position. This setup encourages active participation and collective problem-solving, as the entire class engages directly in the programming process, contributing ideas and solutions in real-time.
